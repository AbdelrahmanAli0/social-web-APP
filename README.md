## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/AbdelrahmanAli0/social-web-APP.git
```

Go to project folder:

```bash
cd social-web-APP
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
